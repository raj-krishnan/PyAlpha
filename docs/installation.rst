==========
Installing
==========

-------------------
Installing with git
-------------------

This project is hosted at https://github.com/raj-krishnan/PyAlpha and can be installed using git:

.. sourcecode:: bash

    $ git clone https://github.com/raj-krishnan/PyAlpha.git
    $ cd PyAlpha
    $ python setup.py install

------------
Dependencies
------------

 - peewee
 - ystockquote
 - pandas
